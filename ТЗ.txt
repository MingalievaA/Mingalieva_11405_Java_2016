        Техническое задание на разработку сайта "Интернет-зоомагазин".

 1. Имя сайта (название домена).
localhost:8080

 2. Название сайта.
Animal Peace

 3. Назначение сайта (цель создания сайта).
3.1. Представление магазина в сети Интернет: информация о животных, цены, контактная информация магазина (или конкретного лица).
3.2. Обеспечение возможности доступа к информации о продаваемых животных, предлагаемых магазином, всем заинтересованным лицам, вне зависимости от их местонахождения.
3.3 Возможность создания объявлений о продаже животных любыми лицами.

 4. Язык сайта.
Русский.

 5. Предполагаемая возрастная аудитория сайта.
Возрастных ограничений нет.

6. Количество страниц сайта.
Сайт должен содержать следующие обязательные html страницы: 
1 - Главная (домашняя) страница; 
2 - Каталог товаров, предлагаемых магазином; 
3 - Информация о товаре, его описание и фотография; 
4 - Страница для обратной связи; 
5 - Корзина товаров; 
6 - Оформление заказа; 
7 - Страница регистрации нового пользователя; 
8 - Страница для аутентификации

7. Кнопки управления (навигация сайта).
Определяются веб-дизайнером самостоятельно.
С каждой страницы сайта должен быть обеспечен переход (установлена гиперссылка) на главную страницу сайта.

8. Блок схема сайта.
Блок схема сайта. 
Головная (начальная) страница сайта должна содержать гиперссылки, обеспечивающие переход с нее на все страницы сайта. 
(логин/регистрация) -> (каталог) -> корзина - Оплата 
-> информация о товаре -> оплата 

9. Оформление рисунков.
Формат всех рисунков png или jpg (jpeg).

10. Основной диапазон разрешения мониторов, на которых будет просматриваться сайт.
Сайт оптимизируется на разрешение в 16:9 пикселей.

11. Основной браузер, которым будет просматриваться сайт, и его минимальная версия.
Chrome

12. Цветовая палитра.
Основной режим мониторов, на которых будет просматриваться сайт: 15 разрядов цветов и выше (число цветов 65536 и выше).
При разработке сайта должен быть обеспечена возможность его просмотра при использовании безопасной цветовой палитры. Изменения оттенков цветов, при просмотре сайта с использованием безопасной цветовой палитры, не оговариваются.

13. Общий фон сайта.
Общий фон сайта светлый (белый). Допускается использование светлого фонового рисунка.

14. Размер и вид шрифта сайта.
Размер шрифта сайта для оформления текста не оговаривается. Размер шрифта для оформления заголовков, названия страниц не оговаривается. Вид (название) шрифта не оговаривается.

15. Срок разработки сайта.
2 месяца.

16. Дополнительные условия.
Каждая страница сайта должна содержать логотип и название магазина. (левый верхний угол)
Внизу на каждой странице сайта должна быть указана контактная информация и гиперссылка для перехода на страницу Обратной связи.
